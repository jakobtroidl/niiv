# Neural Implicit Isotropic Volume Reconstruction

## Getting Started

```
conda create -n niv python=3.9
conda activate niv
conda install pytorch==1.13.1 torchvision==0.14.1 torchaudio==0.13.1 pytorch-cuda=11.7 -c pytorch -c nvidia
```
